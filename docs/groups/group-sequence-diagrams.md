# Group Services Sequence Diagrams

## Create Group Flow

```mermaid
sequenceDiagram
    participant User
    participant Controller as GroupController
    participant Service as GroupService
    participant DB as AppDbContext
    
    User->>Controller: CreateGroupAsync(userId, groupName)
    Controller->>Controller: Extract userId from Claims
    Controller->>Service: CreateGroupAsync(userId, dto)
    
    Service->>Service: Validate name length >= 3
    Service->>Service: Trim whitespace
    
    Service->>DB: Create Group entity
    Service->>DB: Add creator as GroupMember
    Service->>DB: SaveChangesAsync()
    
    DB-->>Service: Confirm saved
    Service-->>Controller: GroupResponseDto
    Controller-->>User: 200 OK with Group details
    
    Note over Service: Auto-generates<br/>InviteToken
```

### Process Steps
1. User sends HTTP request with group name
2. Controller extracts userId from JWT claims
3. Service validates group name (minimum 3 characters)
4. Service creates Group entity with creator as first member
5. Group is persisted to database
6. InviteToken is auto-generated by the database
7. Response returned with group details

## Join Group Flow

```mermaid
sequenceDiagram
    participant User
    participant Controller as GroupController
    participant Service as GroupService
    participant DB as AppDbContext
    
    User->>Controller: JoinGroupAsync(userId, inviteToken)
    Controller->>Controller: Extract userId from Claims
    Controller->>Service: JoinGroupAsync(userId, dto)
    
    Service->>DB: Query Group by InviteToken
    alt Group not found
        Service-->>Controller: NotFoundException
        Controller-->>User: 404 Not Found
    else Group found
        Service->>Service: Check if user already member
        alt Already a member
            Service-->>Controller: ValidationException
            Controller-->>User: 400 Bad Request
        else Not a member
            Service->>DB: Create GroupMember record
            Service->>DB: SaveChangesAsync()
            DB-->>Service: Confirm saved
            Service-->>Controller: OK
            Controller-->>User: 200 OK
        end
    end
```

### Process Steps
1. User sends HTTP request with invite token
2. Controller extracts userId from JWT claims
3. Service queries database for group by invite token
4. If group not found: return 404 NotFoundException
5. If user already a member: return 400 ValidationException
6. Otherwise: create GroupMember record linking user to group
7. Persist to database and return success

## Exit Group Flow

```mermaid
sequenceDiagram
    participant User
    participant Controller as GroupController
    participant Service as GroupService
    participant DB as AppDbContext
    
    User->>Controller: ExitGroupAsync(userId, groupId)
    Controller->>Controller: Extract userId from Claims
    Controller->>Service: ExitGroupAsync(userId, groupId)
    
    Service->>DB: Query Group by Id with Members
    alt Group not found
        Service-->>Controller: NotFoundException
        Controller-->>User: 404 Not Found
    else Group found
        Service->>Service: Find user in Members
        alt Membership not found
            Service-->>Controller: Exception
            Controller-->>User: 400 Error
        else Member found
            Service->>DB: Remove GroupMember
            
            alt Last member?
                Service->>DB: Delete Group
            else Multiple members
                alt User is creator?
                    Service->>Service: Find oldest member
                    Service->>DB: Transfer creator role
                else Not creator
                    Note over Service: No action needed
                end
            end
            
            Service->>DB: SaveChangesAsync()
            DB-->>Service: Confirm saved
            Service-->>Controller: OK
            Controller-->>User: 200 OK
        end
    end
```

### Process Steps
1. User sends HTTP request with group ID
2. Controller extracts userId from JWT claims
3. Service queries group with all members
4. If group not found: return 404 NotFoundException
5. If user not a member: return 400 Exception
6. Remove user from GroupMembers
7. **Special Logic:**
   - If user was last member: delete entire group
   - If user was creator but not last member: transfer owner role to oldest (by JoinedAt) member
   - If user was regular member: no additional action
8. Persist changes and return success
